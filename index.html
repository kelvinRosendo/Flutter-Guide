01
Baixar o Flutter SDK
Acesse o site oficial e inicie o download
›
🌐

Abra o navegador e pesquise por Flutter download no Google, ou acesse diretamente o site oficial:

flutter.dev

No site oficial, clique em Get started para acessar a página de instalação. Em seguida, clique em Quick Start.

💡Sempre use o stable channel para projetos — é a versão mais confiável e testada do Flutter.
02
Selecionar seu Sistema Operacional
As instruções variam por plataforma
›
💻

Na página do Flutter, selecione seu Sistema Operacional. Clique no seu abaixo para ver as dicas específicas:

🪟
Windows
🍎
macOS
🐧
Linux
⚠️Instale em um caminho sem espaços, como C:\src\flutter. Evite C:\Program Files.
  • ◆Baixe o arquivo .zip e extraia na pasta escolhida
  • ◆Adicione o caminho flutter\bin ao PATH do sistema
💡No macOS, você pode usar o Homebrew para facilitar a instalação.
  • ◆Baixe o .zip ou use brew install --cask flutter
  • ◆Adicione o Flutter ao seu ~/.zshrc ou ~/.bashrc
💡No Linux, o Snap é a forma mais fácil de instalar.
  • ◆Use sudo snap install flutter --classic
  • ◆Ou baixe o .tar.xz e configure o PATH manualmente
03
Instalar a Extensão Flutter no VS Code
Configure o editor para desenvolvimento Flutter
›
🔌

Abra o Visual Studio Code. Na barra lateral esquerda, clique no ícone de Extensions (o ícone mais abaixo), ou use o atalho:

Ctrl + Shift + Xabrir extensions

Na barra de busca, digite Flutter e instale a extensão oficial — é o primeiro resultado, com o ícone azul do Flutter.

💡A extensão Flutter instala a extensão Dart automaticamente — ela é necessária para o desenvolvimento.
  • ◆Certifique-se que é a extensão do Dart Code (verificada)
  • ◆Aguarde a instalação completa antes de continuar
04
Criar Novo Projeto via Command Palette
Use Flutter: New Project para iniciar
›
⚡

No VS Code, abra a Command Palette pelo menu View → Command Palette ou pelo atalho:

Ctrl + Shift + Pcommand palette

Na barra de busca que aparecer, digite flutter e selecione a opção Flutter: New Project.

💡Você também pode pressionar F1 para abrir a Command Palette de qualquer lugar no VS Code.
05
Clonar o Flutter SDK
Escolha a pasta e finalize a instalação
›
📥

Ao criar o projeto, o VS Code detectará que o SDK não está instalado e exibirá o botão Download SDK. Clique nele.

Uma janela de seleção de pasta será aberta. Escolha onde instalar o Flutter — sugestão:

C:\src\flutter

Clique em Clonar Flutter e aguarde o download. Pode demorar alguns minutos dependendo da sua internet.

  • ◆A clonagem pode levar 2–5 minutos
  • ◆Após concluir, o VS Code abrirá um novo projeto Flutter
  • ◆Execute o comando abaixo para verificar a instalação:
flutter doctor
🎉Pronto! Pressione F5 no VS Code para rodar o app de exemplo e testar tudo.
🎊
Instalação Concluída!

Seu ambiente Flutter está 100% configurado. Hora de criar algo incrível.

Comandos Essenciais
🩺
flutter doctor
Verifica o ambiente e mostra o que ainda precisa ser configurado.
flutter doctor
▶️
flutter run
Executa o app no emulador ou dispositivo conectado.
flutter run
📦
flutter pub get
Baixa todas as dependências do pubspec.yaml.
flutter pub get
🔄
flutter upgrade
Atualiza o Flutter para a versão estável mais recente.
flutter upgrade
